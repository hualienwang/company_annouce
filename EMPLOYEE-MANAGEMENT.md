# 员工管理功能说明

## 功能概述

管理后台新增了员工管理功能，允许管理员查看、新增和删除公司员工账号。

## 新增功能

### 1. 新增员工按钮
在管理后台顶部快捷操作区域，新增了"👥 新增员工"按钮（绿色背景）。

### 2. 新增员工表单
点击"新增员工"按钮后，会弹出表单，需要填写以下信息：
- 用户名（必填）
- 真实姓名（必填）
- 邮箱（必填）
- 密码（必填，至少6位）

### 3. 员工列表
管理后台新增了"员工列表"区域，显示所有员工的信息：
- 用户名
- 真实姓名
- 邮箱
- 角色（管理员/员工）
- 状态（启用/禁用）

### 4. 删除员工功能
每个员工行的右侧都有"🗑️ 删除"按钮（红色背景），点击后可以删除该员工。

## 使用说明

### 访问管理后台
1. 使用管理员账号登录（admin / admin123）
2. 访问 `/admin` 页面

### 新增员工
1. 点击"👥 新增员工"按钮
2. 填写员工信息
3. 点击"确认创建"

### 删除员工
1. 在员工列表中找到要删除的员工
2. 点击该员工右侧的"🗑️ 删除"按钮
3. 确认删除操作

## 权限说明

- 只有管理员账号才能访问员工管理功能
- 不能删除自己的账号
- 普通员工无法访问管理后台

## 技术实现

### 后端 API
- **删除用户**: `DELETE /api/auth/users/{user_id}`
- **创建用户**: `POST /api/auth/register`（仅管理员）
- **获取用户列表**: `GET /api/auth/users`（仅管理员）

### 前端组件
- **API文件**: `frontend/src/api/auth.ts`
- **页面组件**: `frontend/src/views/Admin.vue`
- **类型定义**: `frontend/src/types/index.ts`

## 注意事项

1. 删除员工后，该员工的所有回复记录将保留在系统中
2. 新建员工默认为"员工"角色，状态为"启用"
3. 用户名和邮箱必须唯一，不能重复
4. 不能删除当前登录的管理员账号
