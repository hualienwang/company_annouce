# 管理后台界面说明

## 正确的访问步骤

### 1. 访问登录页面
```
http://localhost:5000/login
```

### 2. 使用管理员账号登录
- 用户名：`admin`
- 密码：`admin123`

### 3. 登录后，导航栏会显示
```
📢 公司公告   管理后台      [系统管理员]   [退出登录]
```

**注意：只有使用admin账号登录，才会显示"管理后台"链接**

### 4. 点击"管理后台"链接
访问：`http://localhost:5000/admin`

## 常见问题排查

### 问题1：看不到"管理后台"链接
**原因：**
- 没有使用admin账号登录
- 账号权限不足

**解决方法：**
1. 退出登录
2. 使用 admin / admin123 重新登录
3. 登录后导航栏应该显示"管理后台"

### 问题2：看不到新增按钮
**原因：**
- 访问的不是 /admin 页面
- 浏览器缓存问题

**解决方法：**
1. 确保URL是：http://localhost:5000/admin
2. 清除浏览器缓存：Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)
3. 检查浏览器控制台是否有错误（F12打开开发者工具）

### 问题3：看不到删除按钮
**原因：**
- 公告列表为空
- 浏览器缓存问题

**解决方法：**
1. 先创建一些公告（点击"新增公告"）
2. 创建成功后，公告列表会显示删除按钮
3. 清除浏览器缓存：Ctrl+Shift+R

### 问题4：页面显示"暂无公告"
**说明：**这是正常的，说明数据库中还没有公告数据。

**解决方法：**
1. 点击"新增公告"或"新增意见询问"按钮
2. 填写标题和内容
3. 点击"确认发布"
4. 创建成功后，公告列表会显示
